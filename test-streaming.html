Optimizations Implemented

  • requestAnimationFrame: Replaced setInterval with requestAnimationFrame for smoother updates synchronized with browser repaints
  • Adaptive chunk sizing: Dynamically adjusts chunk size based on queue length (10-100 chars per frame)
  • Frame rate throttling: Limits updates to ~60fps to prevent excessive re-renders
  • Simplified streaming rendering: Uses plain text rendering during streaming instead of full ReactMarkdown re-parsing
  • Character batching: Groups characters into optimal chunks for smooth appearance
